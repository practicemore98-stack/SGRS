# Student Grievance Redressal System (SGRS)

A simple **Python console-based project** for managing student grievances using a **CSV file** for data storage.

This project allows students to submit complaints, track grievance status, and enables admin users to update records.

---

## Features

- Submit a new grievance
- Generate unique grievance ID
- Check grievance status using ID
- Admin can update grievance status
- View all submitted grievances
- Stores data in CSV format

---

## Technologies Used

- Python 3
- CSV Module
- OS Module
